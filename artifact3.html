Narrative

Description:
The Animal Rescue Dashboard integrates MongoDB with a Dash/Plotly web interface to visualize live animal shelter data, including animal types, breeds, ages, and locations. This artifact was developed during IT 340: Database Management and enhanced for this ePortfolio.

Justification:
I included this artifact because it demonstrates my database integration skills, query optimization abilities, and understanding of secure data management. Enhancements I implemented include refining database queries for efficiency, improving the user interface for better data interaction, and adding features that highlight selected records on the map. These enhancements show my ability to improve database functionality, maintain data security, and create actionable insights from complex datasets.

Reflection:
Enhancing this dashboard reinforced the importance of efficient database design and secure access, especially when handling potentially sensitive information. Challenges included ensuring fast query responses while maintaining accurate visualizations and preventing unauthorized access to the underlying data. Completing this artifact demonstrates achievement in course outcomes related to database management, professional-quality solution design, and developing a security mindset in data handling.
