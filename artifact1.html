Artifact 1: Android Weight Tracker App (Software Design and Engineering)

Description:
The Android Weight Tracker App is a mobile application that allows users to log their weight over time and view trends via charts. This artifact was originally created in IT 145: Foundations in Application Development.

Justification:
I included this artifact in my ePortfolio because it demonstrates my ability to design functional software with an emphasis on user experience and secure data handling. The app showcases my skills in building intuitive interfaces, implementing data validation, and handling sensitive personal data responsibly. While the original version provided basic functionality, my enhancements focused on improving data security, refining input validation, and creating a more polished user interface, ensuring that the app could handle errors gracefully and store information safely. These improvements highlight my ability to analyze existing software and implement meaningful enhancements.

Reflection:
Enhancing this artifact reinforced my understanding of secure application design and the importance of anticipating potential misuse or errors in user input. I learned to critically evaluate how small design changes can improve both usability and security. Challenges included balancing simplicity with robustness and ensuring that all personal data was protected. With these enhancements, I fully met the course outcomes related to professional-quality software design and development, problem-solving, and security mindset.
